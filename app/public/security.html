Security

Last updated: 06 Apr 2026

We aim to protect user data with reasonable security safeguards. If you believe you’ve found a security issue, please report it responsibly (see “Vulnerability disclosure” below).

1) Encryption in transit

We use HTTPS to protect data in transit between your browser and our servers and between our servers and vendors.

2) Authentication and access controls

  • Authentication: handled via Supabase Auth.
  • Authorization: database access is protected with Row Level Security (RLS) so users access only their own rows.
  • Least privilege: server-side service keys are kept server-only; client uses a public anon key tied to the Supabase project.

3) Payment security (Razorpay)

Premium subscriptions are processed by Razorpay. Payment instruments (cards/UPI/mandates) are handled by Razorpay and are not stored by InHand. We store only identifiers and status needed to enable Premium.

  • Signature verification: we verify Razorpay signatures server-side before granting Premium.
  • Webhooks: when enabled, webhook signatures are validated before processing lifecycle events.

4) Data storage

Saved workspaces and subscription state are stored in Supabase (database). We follow vendor guidance for secure hosting, backups, and access control.

5) Monitoring and logging

Infrastructure providers may maintain standard logs (for example, IP address, timestamps, and request metadata) for reliability and security. We do not intentionally collect advertising identifiers.

6) Incident and breach response

If we suspect a security incident, we take steps to contain it, investigate, and remediate. Where required by law, we will notify affected users and authorities.

7) Vendor management

We use reputable vendors (such as Supabase and Razorpay) for core infrastructure. We review vendor security documentation and configure access keys and permissions carefully.

8) Vulnerability disclosure

Please email security@yourdomain.com with details. Do not publicly disclose vulnerabilities before we have a reasonable opportunity to investigate and fix.
